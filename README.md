# МосГид — путеводитель по Москве

Современный демонстрационный проект на **Next.js 14 (App Router) + Tailwind CSS + shadcn‑стиле компонентов**, который служит основой для интерактивного путеводителя по Москве.

## Структура экранов

- **Главная (`/`)**: геро‑секция с описанием проекта, CTA‑кнопками и блоками‑подборками районов/маршрутов.
- **Карта (`/map`)**: заготовка под основную карту города с фильтрами и списком мест.

Сейчас логика карты и фильтров не реализована — это структурный и визуальный каркас.

## Запуск проекта

```bash
npm install
npm run dev
```

После запуска откройте `http://localhost:3000` в браузере.

## Что можно доработать дальше

- Подключить настоящую карту (Leaflet, Mapbox, Яндекс.Карты).
- Добавить состояние и фильтры (например, через URL‑параметры или глобальный стейт).
- Хранить точки интереса и маршруты в БД или отдельных JSON‑файлах.
